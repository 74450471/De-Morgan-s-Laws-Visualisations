### De Morgan's Law 2

Second law is a bit harder to comprehend than first law since the visualisations of it do not explain it thoroughly. The real idea behind second law is that if an element does not belong to EITHER event A OR event B it does not belong to BOTH element A AND element B. 

In reverse we may look at it in such way: if an element belongs to complements of either event A or event B it necessarily belongs to one of two events only so it does not belong to BOTH event A and event B. 

That is a very easy thing that takes quite a lot of time to understand. I got it only on example that I have provided in the code: if some student is not in either anthropology or mathematical physics then he/she is not in both of them
