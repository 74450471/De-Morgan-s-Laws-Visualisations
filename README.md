### This repository contains different visualisations of mathematical concepts. 

In the first file there are visualisations of De Morgan's Laws. It was for one of my classes on programming in Python taught under L.Kapustina in HSE. One thing I have learned from it is the De Morgan's Law 2 which was a bit harder to understand for visualisation may seem to be deceptive. The idea is that if one element does not belong to event A OR event B, one does not belong to both event A AND event B. I did not understand that EITHER/OR implies that element necessarily belongs to only one of the two events (event A or event B). So if it belongs to only one of them it may not belong to both of them (their intersection).
